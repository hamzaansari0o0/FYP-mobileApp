## ⚡ About The Project

**CourtChuno** is designed to seamlessly connect sports enthusiasts to book arenas, find teammates, and join tournaments. At the same time, it provides arena owners with a powerful, intuitive dashboard to manage their slots, events, and revenues efficiently.

## ✨ Core Features

| Feature | Description |
| :--- | :--- |
| 🔐 **Authentication** | Secure login and signup using Firebase Authentication. |
| 📅 **Real-Time Booking** | Check and book real-time slot availability for sports arenas. |
| 🗺️ **Google Maps** | Easily view ground locations and get directions directly via Google Maps. |
| 🤝 **Player Matchmaking** | Discover and connect with new players near your chosen arena. |
| 💬 **In-App Chat** | Chat directly with other players to organize and schedule matches. |
| 🏆 **Tournaments** | Browse, register, and join exciting sports tournaments. |
| 📊 **Owner Dashboard** | A dedicated and intuitive dashboard for arena owners to manage their slots. |
| 📅 **Event Management** | Owners have the exclusive ability to create and manage their own tournaments. |
| 💳 **Secure Payments** | Integrated secure checkout for payments via Visa and Debit cards. |

## 🛠️ Built With

* **Frontend:** React Native, Expo Router
* **Styling:** Tailwind CSS (twrnc)
* **Backend:** Firebase (Auth, Firestore, Storage)
* **Maps:** Google Maps API
* **Language:** JavaScript (JSX)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js installed. If not, download it from [here](https://nodejs.org/).

### Installation

Run the following commands in your terminal to set up the project:

```bash
# Clone the repository
git clone [https://github.com/hamzaansari0o0/FYP-mobileApp.git](https://github.com/hamzaansari0o0/FYP-mobileApp.git)

# Navigate to the directory
cd FYP-mobileApp

# Install NPM packages
npm install

# Run the App
npx expo start
